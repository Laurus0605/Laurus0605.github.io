云计算实验3实验报告

学号：202200130110 姓名：李彤 班级：计机22.2
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 2025年3月12日
实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。
硬件环境： 联网的计算机一台
软件环境：Windows or Linux

实验步骤与内容：

1.注册github账号，创建新的仓库

2.安装node.js软件

​ 直接在node.js官网安装即可，安装完成后如图所示

​ 输入“node -v”命令验证是否正确安装node.js

3.在本地文件夹中创建一个用于存放github本地仓库文件的目录

4.下载hexo并初始化

​ 右击blog文件夹，点击更多选项，然后选择“Git Bash Here”选项，然后输入cnpm install -g hexo-cli命令安装hexo，然后输入“hexo init”命令对当前文件夹进行初始化

​ 输入“hexo v”命令查看hexo版本验证安装成功

5.安装hexo服务器

输入“npm install”命令和“npm install hexo-server –save”命令安装hexo服务器

输入“npx hexo server”命令启动服务器，现在在浏览器中打开网址即可进入网页，输入Ctrl+C即可终止服务器，结果如图：

6.初步测试

输入“hexo g”命令生成静态网页，如图所示：

bash中输入“hexo new post ‘上传测试’”来发布新帖子

7.生成SSH

在bash中输入输入git config –global user.name “userrname”和git config –global user.email “useremail”，然后输入git config –list命令检查用户名和邮箱是否正确。

输入“ssh-keygen -t rsa -C “youremail”命令生成密钥，对于所有问题，全部选择回车。然后在文件夹C:\Users\lenovo.ssh下找到id_rsa.pub文件，获得密钥。

在github中找到settings，然后找到SSH keys，点击new SSH key 新建密钥，如图所示：

然后输入ssh -T git@github.com，如果显示拒绝连接的字样，则输入“vim ~/.ssh/config”命令进入编辑器，输入“Host github.com Hostname ssh.github.com Port 443”命令后，再次尝试连接，即可成功，如图所示：

8.将项目部署到Github

修改blog中的_config.yml文件，添加如图所示蓝色字样：

在bash中输入npm install hexo-deployer-git –save安装工具，然后输入hexo g 和 hexo d命令将项目推送到Github，如图所示，输入hexo d命令后会报错：

修改_config.yml中对应的蓝色字样后即可成功推送，如图所示：

在仓库中打开设置，点击Settings，在左侧一栏选择Pages，打开个人站点即可打开网站https://laurus0605.github.io/，如图所示，网站已更新

9.将本次报告上传至个人网站，如本网站所示

结论分析与体会

结论分析：

在本次实验中通过Github、Git、node.js、hexo以及Typora五者合作完成了全部实验内容，首先注册个人的Github账号，然后下载Git和node.js，新建文件夹利用Git打开，然后即可下载hexo的相关内容，利用hexo创建个人静态网站，启动服务器后即可打开。可以通过Typora编辑markdown文件，即修改自己的博客内容，将Github与本地连接后，即可将个人博客推送到网站。

心得体会：

通过本次实验，学习到了如何利用Github以及hexo构建自己的个人网站并上传自己的博客。在实验的过程中，每一步都是很新奇的体验，从建造一个空白的个人网站，再到第一篇测试博客的成功推送，最后成功编辑自己的实验报告并完成推送，每一步都很有成就感。
